# 🎮 RPG in C

Projeto desenvolvido em linguagem C como forma de comemorar o encerramento do meu primeiro período da graduação em Sistemas de Informação na Universidade Federal de Viçosa (UFV).

O objetivo deste projeto é reunir os conhecimentos adquiridos durante os primeiros meses da faculdade em um único programa, aplicando conceitos de lógica de programação, estruturas de dados, modularização, funções, structs e organização de código.

## 📖 Sobre o Projeto

Este é um RPG desenvolvido em C que está sendo construído gradualmente conforme novos conhecimentos são adquiridos ao longo da graduação.

Além de servir como um desafio pessoal, o projeto representa minha evolução como estudante de tecnologia, colocando em prática conteúdos aprendidos em sala de aula e explorando novas funcionalidades além das exigidas nas disciplinas.

## 🚀 Objetivos

- Praticar programação em C.
- Aplicar conceitos vistos no primeiro período.
- Melhorar a organização e modularização de código.
- Desenvolver lógica de programação através de um projeto real.
- Registrar minha evolução acadêmica e técnica.

## 🛠️ Tecnologias Utilizadas

## 📂 Estrutura do Projeto

O projeto está organizado em múltiplos arquivos para facilitar a manutenção e evolução do código.

```text
RPG-in-c/
├── main.c
├── ...
```

## ▶️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/shtMCe/RPG-in-c.git
```

Entre na pasta do projeto:

```bash
cd RPG-in-c
```

Compile o código:

```bash
gcc *.c -o rpg
```

Execute:

```bash
./rpg
```
